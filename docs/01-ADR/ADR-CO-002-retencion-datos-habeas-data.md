# ADR-CO-002: Retención y trazabilidad de consentimiento (Habeas Data)

## Estado
Aceptado

## Contexto
La normativa local exige controles sobre autorización, conservación y ejercicio de derechos de titulares.

## Decisión
Mantener evidencia de consentimiento, políticas de retención y proceso de atención de derechos.

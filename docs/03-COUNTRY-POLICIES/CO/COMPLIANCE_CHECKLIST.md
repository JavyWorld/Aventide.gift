# CO - Compliance Checklist

- [ ] **Fiscal:** reglas tributarias locales vigentes por tipo de operación.
- [ ] **Facturación:** emisión y validación de factura electrónica en flujo productivo.
- [ ] **Privacidad:** consentimiento y finalidades registradas para tratamiento de datos.
- [ ] **Retención:** política de retención/borrado alineada con obligaciones legales.
- [ ] **Payouts:** validación de beneficiarios y controles de liquidación.
- [ ] **Disputas:** gestión y evidencia de reclamos dentro de SLA definidos.

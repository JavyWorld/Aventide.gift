# CO (Colombia) - Alcance local

## Alcance regulatorio
- Facturación electrónica conforme a lineamientos DIAN.
- Tratamiento tributario local para servicios y comisiones.
- Exigencias de privacidad y Habeas Data sobre datos personales.

## Alcance operativo
- Integración/flujo de validación de factura electrónica.
- Parametrización fiscal local para cálculo y reporte.
- Gestión de consentimientos y derechos de titulares.

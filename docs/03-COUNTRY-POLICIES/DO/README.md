# DO (República Dominicana) - Alcance local

## Alcance regulatorio
- Tratamiento fiscal local para servicios digitales y/o marketplace.
- Emisión de comprobantes conforme a obligaciones locales aplicables.
- Requisitos de retención sobre pagos a proveedores locales.

## Alcance operativo
- Parametrización de impuestos y retenciones por tipo de actor.
- Soporte de evidencia fiscal para auditoría y conciliación.
- Gestión de disputas y reversos bajo tiempos internos definidos para DO.

# DO - Compliance Checklist

- [ ] **Fiscal:** reglas de ITBIS activas por categoría y contribuyente.
- [ ] **Facturación:** comprobantes emitidos con datos fiscales locales requeridos.
- [ ] **Privacidad:** base legal documentada para tratamiento de datos personales.
- [ ] **Retención:** políticas de conservación de evidencia fiscal y contractual vigentes.
- [ ] **Payouts:** retenciones aplicadas según perfil fiscal del beneficiario.
- [ ] **Disputas:** proceso local de reclamos/reversos con SLA documentado.
